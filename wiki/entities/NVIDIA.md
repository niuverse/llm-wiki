# NVIDIA

NVIDIA 是 [[RoboLab]] paper/project/repository 的发布机构，也是 [[isaac-sim-asset-structure|Asset Structure - Isaac Sim Documentation]] 的 publisher。RoboLab source 中相关的技术上下文是 NVIDIA Isaac Lab、Isaac Sim、Omniverse EULA、NVlabs GitHub organization，以及 Simulation and Robotics Lab project page；Asset Structure source 则把 NVIDIA Isaac Sim 6.0 EDR 的 robot asset organization 写成 USD layers、payloads、references 和 variants。这里的 entity page 只记录与本 wiki sources 相关的信息，不扩展到 NVIDIA 的 broader corporate history。
