# Pixar

Pixar Animation Studios 是 [[openusd-introduction|Introduction to USD]] 的 copyright holder。该 source 还把 USD 放在 Pixar production pipeline 的历史中：USD 是 Pixar 3D graphics pipeline 的核心，源自 Pixar 多代 composed scene description system 的演化，目标是把 Presto 的 composition engine / low-level data model 与 TidScene 的 lazy-access、time-sampled data model 和 lightweight scenegraph 结合起来。

这里的 entity page 只记录 Pixar 与已 ingest OpenUSD official docs 的关系，不扩展到 Pixar 的 broader company history。

相关页面：[[OpenUSD]]、[[OpenUSDSceneComposition]]。
